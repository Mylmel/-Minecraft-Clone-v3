# Редактор модов для Minecraft Clone

## Обзор

Редактор модов позволяет создавать, редактировать и управлять модификациями для игры Minecraft Clone прямо во время игры. С помощью редактора вы можете:

- Создавать новые блоки с уникальными текстурами и свойствами
- Добавлять новые предметы в игру
- Определять рецепты крафта для новых и существующих предметов
- Добавлять новые сущности (в будущих обновлениях)

## Как использовать

### Открытие редактора

Для открытия редактора модов нажмите `Ctrl+M` во время игры.

### Интерфейс редактора

Редактор модов имеет интуитивный интерфейс с следующими разделами:

1. **Информация о моде** - основные данные о моде:
   - ID мода: уникальный идентификатор (латинские буквы, цифры, дефисы)
   - Название мода: отображаемое имя мода
   - Версия: номер версии в формате X.Y.Z
   - Автор: ваше имя или ник
   - Описание: краткое описание мода

2. **Добавление блоков** - создание новых блоков:
   - Название блока
   - Текстура (путь к файлу текстуры)
   - Твердость (влияет на время добычи)

3. **Добавление предметов** - создание новых предметов:
   - Название предмета
   - Текстура
   - Максимальное количество в стаке

4. **Добавление рецептов** - определение способов крафта:
   - Тип рецепта (крафтинг или выплавка)
   - Входные материалы (ID, количество)
   - Выходной результат (ID, количество)

5. **Добавление сущностей** - создание новых существ (в разработке)

### Горячие клавиши

- `Ctrl+M` - открыть/закрыть редактор модов

## Программное использование

Редактор модов также можно использовать программно в коде:

```typescript
import { modEditor } from './mod-editor';

// Создать новый мод
const myMod = modEditor.createNewMod('my-mod', 'Мой мод', '1.0.0', 'Автор', 'Описание');

// Добавить блок
const diamondBlockId = modEditor.addBlockToCurrentMod('Алмазный Блок', 'diamond.png', 5);

// Добавить предмет
const diamondId = modEditor.addItemToCurrentMod('Алмаз', 'diamond-item.png', 64);

// Добавить рецепт
modEditor.addRecipeToCurrentMod('crafting', [{id: diamondId, count: 9}], {id: diamondBlockId, count: 1}, true);

// Сохранить мод
modEditor.saveCurrentMod();
```

## Интеграция с игрой

После добавления редактора модов в игру, он автоматически:

- Добавляет новые блоки и предметы в соответствующие списки
- Интегрирует новые рецепты в систему крафта
- Позволяет получать новые предметы через команду `/give`
- Обеспечивает корректное отображение новых блоков и предметов в инвентаре

## Формат JSON модов

Моды могут быть экспортированы и импортированы в формате JSON. Пример структуры:

```json
{
  "id": "example-mod",
  "name": "Примерный Мод",
  "version": "1.0.0",
  "author": "Developer",
  "description": "Это пример мода",
  "blocks": [
    {
      "id": 100,
      "name": "Алмазный Блок",
      "texture": "diamond_texture.png",
      "hardness": 5,
      "toolRequired": "stone_pickaxe",
      "dropId": 100,
      "dropCount": 1
    }
  ],
  "items": [
    {
      "id": 200,
      "name": "Алмаз",
      "texture": "diamond_item.png",
      "maxStack": 64
    }
  ],
  "recipes": [
    {
      "type": "crafting",
      "input": [{"id": 200, "count": 9}],
      "output": {"id": 100, "count": 1},
      "craftingTable": true
    }
  ],
  "entities": []
}
```

## Совместимость

Редактор модов использует ID начиная с 100 для блоков, 200 для предметов и 300 для сущностей, чтобы избежать конфликта с оригинальными ID в игре.
